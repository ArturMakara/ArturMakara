# BUD-ART AI / Modlan Systems 🛠️

Rozwiązania IT i automatyzacja procesów dla firm z branży telekomunikacyjnej i produkcyjnej. Działamy na Podkarpaciu — łączymy doświadczenie wykonawcze (budowa sieci FTTH, modernizacje LAN, meble na wymiar) z własnym software'em.

## 🏗️ Czym się zajmujemy

- **Budowa sieci światłowodowych FTTH** — projektowanie (QGIS), wykonawstwo, dokumentacja powykonawcza
- **Modernizacje sieci LAN** — instalacje teletechniczne i elektryczne, pomiary certyfikowane
- **Meble na wymiar** — marka [Równa Linia](https://www.rownalinia.pl): produkcja, przetargi publiczne, klienci indywidualni

## 💻 Nasze narzędzia własne

- **Systemy webowe (Python/Flask)** — rejestracja czasu pracy ekip terenowych, zarządzanie flotą pojazdów, portale firmowe
- **Automatyzacja workflow (n8n, Apps Script)** — obsługa zgłoszeń awarii sieci, powiadomienia SMS/e-mail, eskalacje
- **Integracje API** — KSeF (Krajowy System e-Faktur), Todoist, SMSAPI
- **GIS** — analiza przestrzenna sieci, generowanie map i dokumentacji projektowej (PyQGIS, ezdxf)
- **Raportowanie** — kosztorysy, rozliczenia, dokumentacja przetargowa (openpyxl)

## 🧰 Technologie

`Python` `Flask` `n8n` `PyQGIS` `Google Cloud` `nginx` `Google Apps Script`
